Résumé

Pourquoi une ville se structure ainsi? Est-ce une forme optimale? Comment se développe-t-elle?

Introduction / Intérêt de la recherche

A travers un modèle viable, nous voulons analyser les relations spatiales entre les différents composants tels que les habitations, les lieux d'activités et le vide actif. Cette analyse devra permettre de mieux comprendre les interactions entre différents paramètres au sein d'une ville.

Objectifs et hypothèses

Pour ce faire, nous avons définis des unités représentant des éléments structurels types d'une ville. Ainsi, une cellule rouge représente une unité d'habitation type, une cellule verte, une unité d'activité et une cellule inactive, le vide actif. Le nombre d'unités est paramétrable.

Nous avons émis la condition qu'une cellule d'habitation doit avoir dans un rayon donné r un nombre d'activités nb pour être stable. Si la condition n'est pas satisfaite, l'initialisation est réitérée jusqu'à ce que la condition soit satisfaite..
